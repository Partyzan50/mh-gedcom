Частые вопросы

Работает ли на чужих деревьях?

Да. Букмарклет использует BFS-обход через API родственников — он доступен для любого дерева, которое вы можете видеть. Персоны <Private> будут в файле, но без имён.

Почему кнопку нельзя просто кликнуть здесь?

Букмарклет должен выполниться на странице MyHeritage, а не здесь — иначе у него не будет доступа к данным дерева. Именно для этого нужно перетащить его в закладки и кликнуть там.

Что делать если появляется ошибка 429?

Букмарклет автоматически делает паузу 6–9 секунд и повторяет запрос. Если ошибки продолжаются — нажмите «Пауза», подождите 2–3 минуты, затем «Продолжить».

В каких программах открыть .ged файл?

Gramps (бесплатно), Древо Жизни, Family Tree Maker, MacFamilyTree, Ancestris, Legacy Family Tree — любая программа поддерживающая GEDCOM 5.5.1.

Данные передаются на сервер?

Нет. Весь код работает только в вашем браузере. Запросы идут только к MyHeritage, GEDCOM файл создаётся локально.
