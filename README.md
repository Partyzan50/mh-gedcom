# 🌳 MyHeritage → GEDCOM v15

Букмарклет для экспорта **любого** дерева MyHeritage в файл `.ged` (GEDCOM 5.5.1).

## 🔗 Страница установки

**https://partyzan50.github.io/mh-gedcom/**

Перетащите зелёную кнопку на панель закладок.

## Файлы репозитория

| Файл | Назначение |
|------|-----------|
| `index.html` | Страница установки с кнопкой и форумными сниппетами |
| `bookmarklet.js` | Сам скрипт (загружается при клике на закладку) |
| `README.md` | Этот файл |

## Возможности

- ✅ Своё и чужое дерево
- ✅ Имена, даты рождения/смерти
- ✅ Семейные связи (супруги, дети, родители)
- ✅ Кнопка паузы и возобновления
- ✅ Скачать в любой момент
- ✅ Защита от 429 (Too Many Requests)
- ✅ Всё в браузере — без серверов
- ✅ GEDCOM 5.5.1

## Как использовать

1. Откройте [https://partyzan50.github.io/mh-gedcom/](https://partyzan50.github.io/mh-gedcom/)
2. Перетащите зелёную кнопку на панель закладок
3. Откройте любое дерево на MyHeritage
4. Кликните закладку → появится виджет
5. ⏸ Пауза / ▶ Продолжить / ↓ Скачать GEDCOM
